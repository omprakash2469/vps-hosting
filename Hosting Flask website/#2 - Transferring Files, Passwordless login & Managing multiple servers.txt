Enable password less login to your server

1. Create a private-public key pair
	-  ssh-keygen -t rsa

2 - Deploy ssh key on your server
	- cd ~
	- mkdir .ssh

3. Enter command on your personal computer
	- scp C:\Users\Omprakash\.ssh\id_rsa.pub username@165.232.177.116:~/.ssh/authorized_keys

4. Test your passwordless login
	- ssh user@ip-address

5. Create powershell profile
	-  New-Item $profile -Type File -Force
	- notepad $profile
