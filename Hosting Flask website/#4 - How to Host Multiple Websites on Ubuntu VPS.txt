1. Add 'A' Record in your DNS Management
2. Point your domain to your vps server

1 - Creating Directories for individual sites
	- sudo mkdir -p /var/www/yoursite.com/

2 - Transfer the site contents

3 - Creating the VirtualHost files
	-  sudo vim /etc/apache2/sites-available/yoursite.com.conf

4 - Enter this content in .conf file
<VirtualHost *:80>
    ServerName yoursite.com
    ServerAdmin yourPublicEmail@email.com
    DocumentRoot /var/www/yoursite.com
    ErrorLog ${APACHE_LOG_DIR}/error.log
    CustomLog ${APACHE_LOG_DIR}/access.log combined
</VirtualHost>

4 - Enable the VirtualHosts
	- cd /etc/apache2/sites-available/
	- sudo a2ensite yoursite.com.conf
	- sudo service apache2 restart
